# Welcome to Arduino Week

Welcome to the Arduino Week documentation! Over the next four days, you'll go from knowing nothing about microcontrollers to building your own interactive electronic projects.

This workshop series is designed to be **hands-on, practical, and beginner-friendly**. Each day builds on the previous one, taking you through a carefully structured learning path.

## What You'll Learn

### Day 1: Arduino Basics
**Focus:** Fundamentals of Arduino programming and digital I/O

You'll learn:
- How to set up the Arduino IDE and connect your board
- Understanding Arduino pins and their functions
- Core programming concepts (setup, loop, variables)
- Building circuits with LEDs and buttons
- Reading digital inputs and controlling outputs

**Project:** Button-controlled LED with toggle functionality


### Day 2: Ultrasonic Security System
**Focus:** Proximity sensing and system integration

### Day 3: Mini Piano
**Focus:** Sound synthesis and multiple button inputs

### Day 4: IEEE x AP-S Arduino Antenna
**Focus:** RF foundations and antenna measurements

---

## Workshop Structure

Each day follows the same format:

1. **Introduction** - Learn the concepts and theory
2. **Hands-on Building** - Follow along as we build circuits together
3. **Code Walkthrough** - Understand how the code works step-by-step
4. **Challenge** - Apply what you've learned to solve problems

---

### Prerequisites
**Absolutely none!** This workshop assumes no prior knowledge of:
- Programming
- Electronics
- Circuit building

If you can use a computer and are willing to learn, you're ready to go.
